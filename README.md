# JSP_TeamProject

JSP 기반 3개 게시판(공지, 자유, 자료실) 팀 프로젝트입니다.  
IntelliJ + Tomcat 9 + MySQL 환경에서 실행 가능합니다.

## 📂 브랜치 구조
- main : 최종 배포용
- develop : 통합용
- feature/notice : 공지사항 담당
- feature/free : 자유게시판 담당
- feature/file : 자료실 담당

## ⚙️ 실행 방법
1. MySQL DB 생성 (teamdb)
2. Tomcat 9 서버 등록
3. http://localhost:8080/index.jsp 실행
